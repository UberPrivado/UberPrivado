## Cómo publicar en GitHub Pages

### Paso 1 — Crear repositorio en GitHub
1. Ir a [github.com](https://github.com) e iniciar sesión
2. Clic en **New repository**
3. Nombre: `transporte-privado-premium` (o el que prefieras)
4. Dejarlo en **Public**
5. **NO** marcar "Add a README file"
6. Clic en **Create repository**

### Paso 2 — Subir los archivos
En la página del repositorio vacío, clic en **uploading an existing file** y sube todos los archivos:
- `index.html`
- `style.css`
- `script.js`
- `ChatGPT Image 24 may 2026, 05_08_59 p.m..png`
- `Gemini_Generated_Image_ctq9icctq9icctq9.png`
- `.nojekyll`
- `README.md`
- `package.json`

Clic en **Commit changes**.

### Paso 3 — Activar GitHub Pages
1. Ir a **Settings** del repositorio
2. En el menú lateral clic en **Pages**
3. En **Source** seleccionar: **Deploy from a branch**
4. En **Branch** seleccionar: **main** → carpeta **/ (root)**
5. Clic en **Save**

### Paso 4 — Ver tu sitio publicado
Después de 1-2 minutos tu sitio estará disponible en:
```
https://TU_USUARIO.github.io/transporte-privado-premium/
```
